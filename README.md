# My very first web page

Welcome! [Code Institute](https://codeinstitute.net)

## What is available?

This website highlights the different streams available to learners at Code Institute. The aim of the website is to highlight basic use of the HTML language & CSS styles inline with use of GitHub and an accompanying README.md file.

Code Institute is the leading global provider of online coding education. Credit-rated by Edinburgh Napier University and validated by the Ministry of Education in Ontario, the Higher Education Authority in Sweden, the Ministry of Communications in Singapore, and the Higher Education Authority in Ireland.

Click the links at the top to see Wikipedia details of the courses available on 3 different streams so you can make an informed choice about which stream is most suited to you.

## Streams

Over the course of 6 months you can choose to enter one of 3 streams (links provided) dependant upon your interests or requirements. Course content for each page is as follows:

### Stream One Page Links

1. HTML
2. CSS
3. JavaScript
4. Cloud9
5. Git (Introduction)
6. GitHub (Introduction)

### Stream Two Page Links

1. Python (Intermediate)
2. Flask
3. MongoDB
4. MySQL (Intermediate)
5. Git (Intermediate)
6. GitHub (Intermediate)

### Stream Three Page Links

1. Python (Advanced)
2. Django
3. Heroku
4. Git (Advanced)
5. GitHub (Advanced)
6. MySQL (Advanced)

Clickable links to Wikipedia sites for each Stream content which include, languages, web-frameworks, cloud platforms, version control, hosting and database management.
